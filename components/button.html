    # Button 按钮

    TIP

    常用的操作按钮。

    # 基础用法


    基础的按钮用法。
    type 接收 default、success、primary 字符串类型。
    <Button>default</Button>
<Button type="success">success</Button>
<Button type="primary">primary</Button>
    显示代码

    # 不同尺寸


    Button 组件提供三种尺寸，可以在不同场景下选择合适的按钮尺寸。

    size 接收 default、small 和 large 三种尺寸。
    <Button size="large">large </Button>
<Button>default </Button>
<Button size="small">small </Button>
    显示代码

    # 按钮组


    以按钮组的方式出现，常用于多项类似操作。

    ButtonGroup 标签来嵌套你的按钮
    <ButtonGroup>
    <Button>Create new file</Button>
    <Button>
      Upload files
    </Button>
    <Button>Find file</Button>
</ButtonGroup>
    显示代码

    # props


    参数 说明 类型 可选值 默认值
    type 按钮类型 string default,primary,success default
    size 按钮大小 string small, default, large default
