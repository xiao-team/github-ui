    # Page 分页

    TIP

    常用的分页组件。

    # 基础用法


    基础的分页用法。
    Previous 1 2 3 … 17 18 Next
    type 接收 default、success、primary 字符串类型。
    <Page :total="180"></Page>
    显示代码

    # props


    参数 说明 类型 可选值 默认值
    total 总数 number - 0
    pageSize 每页数量 number -- 10
    page 当前页 number -- 1
    prevText 上一页文案 string -- Previous
    nextText 下一页文案 string -- Next

    # event


    事件名 说明 返回值
    pageChange 页码变化时触发的事件 当前页码
